# SmartHMM

A open-source HRM system built with:

- **Go** (backend API)
- **Next.js** (frontend dashboard)
- **Monorepo** structure for easy development

---

## Tech Stack

- **Backend:** Go 
- **Frontend:** Next.js
- **Database:** PostgreSQL
